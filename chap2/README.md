# Code from CSAPP Chapter 2

## Compile
Compile show_bytes.c into a 32bit binary and double check.
```
gcc -o show_bytes -m32 show_bytes.c
file show_bytes
```

## Experiment
Edit and run show_bytes to study the bit representations of various data types.
```
./show_bytes
./show_bytes 1
./show_bytes -1
```
